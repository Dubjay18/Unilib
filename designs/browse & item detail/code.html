The Elements of Computing Systems

Building a Modern Computer from First Principles

Noam Nisan, Shimon Schocken

star star star star star_half 4.5 (128 Reviews)
| Published: 2005 (MIT Press) | ISBN: 9780262640688

This textbook provides an integrated and rigorous introduction to computer science, building a complete hardware platform and software hierarchy from the ground up.

Students will learn how to design logic gates, build a CPU, develop an assembler, a virtual machine, a compiler, and finally, a basic operating system, gaining a profound understanding of how computers actually work.
